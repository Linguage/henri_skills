# 目录结构参考模板

Downloads 文件整理的完整目录结构，含分类说明。

## 完整目录树

```
Downloads/
├── 书籍/                          # PDF、EPUB 等书籍文件
│   ├── (主题类别A)/               # 按用户研究方向/兴趣自定义
│   ├── (主题类别B)/
│   ├── (主题类别C)/
│   ├── 书籍清单.md                 # 所有文档的可查验索引（含中文译名+链接）
│   ├── inventory.json             # 主数据库（唯一数据源，MD由此生成）
│   └── (其他类别/)                 # 根据实际内容讨论确定
├── 文档/                          # PPTX、XLSX、DOCX、TXT、HTML、JSON 配置等
├── 图片/                          # 各类图片文件（三层分类系统）
│   ├── AI生成图片/                 # ChatGPT、Gemini 等生成图片
│   │   ├── ChatGPT/
│   │   └── Gemini/
│   ├── 社交平台图片/               # 微信、Telegram、Twitter、知乎等
│   │   ├── 微信/
│   │   ├── Telegram/
│   │   ├── Twitter/
│   │   └── 知乎/
│   ├── 手机拍摄/                   # 手机/相机拍摄照片（含 EXIF 证据）
│   ├── 截图/                      # 手机/电脑截图、剪贴板粘贴
│   ├── 网络下载图片/               # 壁纸、文章配图、缩略图、插画
│   │   ├── 壁纸/
│   │   └── YouTube缩略图/
│   ├── 学术与工程图片/             # 研究图表、工程图示
│   ├── (其他 AI 平台)/             # 按实际使用的 AI 工具建子目录
│   └── 其他图片/                  # 未匹配分类的图片（兜底）
├── 视频/                          # MP4、MOV 等视频文件
├── 安装包/                        # DMG、PKG、ZIP 安装文件
├── 数据文件/                      # CSV、JSON 数据文件
├── 个人档案/                      # 敏感个人信息文档
├── 音频/                          # M4A、MP3 等音频文件
├── 临时文件/                      # CRDOWNLOAD 等临时文件
└── 待删除/                        # 需要用户确认后删除的文件
```

## 个人档案子目录

```
个人档案/
├── 身份证件/                 # 身份证、护照、驾驶证等
├── 学籍与证明/               # 学生证、登记表、成绩单、毕业证
├── 财务与法律/               # 银行流水、合同、法院文件
├── (个人项目)/               # 按项目名建子目录
├── (AI 工具聊天归档)/        # claude/chatgpt 对话导出
├── (社交媒体平台)/           # 按平台名建子目录
└── (其他)                    # 无法归入上述类别的敏感文件
```

## 文档子目录（按需自定）

```
文档/
├── (专题A)/                  # 按内容专题建子目录
├── (专题B)/
├── (已有专题项目)/           # 已有的专题文件夹保持不动
└── (散落文件)                # 无法归类的保留在根目录
```
